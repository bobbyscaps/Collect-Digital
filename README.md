# Collect Digital

NFT project intelligence platform.

Evaluate NFT projects using:
- Floor price
- Holder count
- Sales
- Liquidity
- Community strength
- Founder influence
- Revenue generation
- IRL experiences
- Art quality
- Longevity
- Reward systems
- Wallet behavior

Features:
- Collection ratings
- Wallet collector score
- Flipper badges
- Community-funded evaluations
- Portfolio valuation
- Sell alerts
- Wiki pages
- OpenSea integration

Status:
🚧 Early development

Vision:
Become the IMDb + CoinMarketCap + Wikipedia + Floor App for NFT communities.
